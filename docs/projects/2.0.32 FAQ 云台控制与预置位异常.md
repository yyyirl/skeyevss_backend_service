## 1. 控制链路径

云台指令一般经历：**客户端 → 平台 API → 信令模块 → 设备**。任一环节 **超时、鉴权失败、通道 ID 映射错误** 都会表现为「按钮无反应」。先确认 **通道是否在线**、**用户是否有控制权限**（见权限模型文档）。

---

## 2. 国标与厂商差异

- **GB28181** 中 PTZ 常通过 **MESSAGE** 或扩展流程承载，设备实现 **参差不齐**；
- 部分设备 **仅 ONVIF 支持 PTZ**，国标只出图；
- **预置位编号** 各厂从 0 或 1 起始，平台映射错误会导致「调用预置位到了错误角度」。

---

## 3. 常见故障

| 现象 | 可能原因 |
|------|----------|
| 连续点动后失控 | 未发停止命令、设备防抖差 |
| 预置位漂移 | 机械云台未校准、断电丢记忆 |
| 偶发 408 | 网络抖动、设备忙、并发控制冲突 |
| 仅管理员可用 | RBAC 未授权普通角色 |

---

## 4. 并发与互斥

同一通道 **同时** 多用户控制需 **串行化** 或 **抢占**；否则设备固件可能 **丢弃指令** 或执行乱序。平台侧应有 **队列与超时**，并向 UI 反馈 **忙状态**。

---

## 5. 日志与复现

记录：**用户、通道 ID、指令类型、时间戳、设备响应码**；复现时固定 **单用户单通道**，排除并发干扰。若直连设备 Web 正常而经平台失败，重点查 **平台到设备的信令映射**。

---

云台问题 **权限 + 映射 + 设备能力** 三角；不要在国标设备上假设 ONVIF 级别的 PTZ 一致性。
